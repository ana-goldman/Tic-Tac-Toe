## Spec

### Step 1 — Visual Tone
- [x] Establish a polished minimal visual foundation using soft neutral colors and muted accents.
The palette should feel calm, contemporary, and refined rather than playful or high-contrast. Use spacing, typography, and restrained color choices to create a clean interface that feels intentionally designed.

### Step 2 — Board Depth
- [x] Style the tic-tac-toe board with subtle depth, soft shadows, and carefully balanced cell boundaries.
The board should feel tactile without becoming heavy or decorative. Shadows and surface colors should gently separate the play area from the page while preserving the minimal overall look.

### Step 3 — Drawn Marks
- [x] Create minimal hand-drawn X and O marks inspired by contemporary Japanese styling.
The marks should feel organic and ink-like while staying clean enough for fast readability. Use refined stroke weight, muted accent colors, and simple mark shapes that avoid cartoonish or overly ornate details.

### Step 4 — Calm Motion
- [ ] Add tiny transitions, tactile move feedback, and subtle ink-brush inspired win motion.
Interactions should feel calm and satisfying, with short animations that support the game state without distracting from play. Move placement, hover states, and win feedback should all share the same restrained motion language.
